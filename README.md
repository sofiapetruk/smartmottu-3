## Como Rodar o Projeto

    ### Pré-requisitos
        IntelliJ IDEA
        JDK-17
        Oracle 11
### Clonar repositorio

    https://github.com/sofiapetruk/smartmottu-3.git

### Abra o projeto no IntelliJ IDEA:

    Inicie o IntelliJ

    Vá em File > Open... e selecione a pasta do projeto clonado

    Aguarde a indexação e o carregamento do Maven

### Execute o projeto:

    Clique na seta verde ▶ no canto superior direito

    Ou use o atalho Shift + F10

### Como entrar no projeto
    1. Achar a classe main
        **Caminho:** src > main > java > br.com.fiap.smartmottu > SafehubApplication
    2. Rodar a classe main com o botão "play" ao lado superior direito da idea
    3. localhost:8080/login
    4. Faça o cadastro do seu usuário **OU** use Usuário ADMIN
            - email: admin@email.com
            - senha: admin123
